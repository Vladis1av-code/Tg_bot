## 📋 Про проєкт

Цей Telegram-бот надає актуальну інформацію про відключення електроенергії в містах Миколаївської області. Дані отримуються в реальному часі через API з офіційного сайту АТ «Миколаївобленерго» — [off.energy.mk.ua](https://off.energy.mk.ua/).

Бот підтримує всі основні міста області та дозволяє обрати конкретну чергу відключення, щоб дізнатися поточний стан без зайвих кліків.

---

## 🌟 Можливості

- 🔌 **Перевірка статусу** — дізнайтеся, чи є зараз відключення у вашій черзі
- 🗺️ **Вибір міста** — підтримка усіх міст Миколаївської області
- ⚡ **Вибір групи** — групи від 1.1 до 6.2 (ГПВ, ГАВ, СГАВ)
- 📡 **Актуальні дані** — інформація береться безпосередньо з офіційного джерела через API
- 🛠️ **Технічна підтримка** — вбудована форма звернення до команди
- 📢 **Масова розсилка** — адміністратор може надсилати повідомлення всім користувачам

---

## 🖥️ Як це працює

1. Користувач обирає `Миколаївська область`
2. Обирає місто, наприклад `Новий Буг`
3. Обирає групу, наприклад `Група 5.2`
4. Бот відповідає: ✅ *Зараз відключень немає* або 🔴 *Відключення активне*

---

## 👥 Ролі користувачів

### 👤 Користувач (User)
- Перевірка графіку відключень по місту та групі
- Надсилання звернень до технічної підтримки

### 🛡️ Адміністратор (Admin)
- Усі дії звичайного користувача
- `/place_admin` — вибрати місто
- `/send` — розіслати повідомлення всім користувачам
- `/stats` — статистика використання
- `/website_admin` — адміністрування сайтів
- `/dev` — інформація про розробників

### 👑 Власник (Owner)
- Усі права адміністратора
- `/add_admin ID` — додати адміністратора
- `/remove_admin ID` — видалити адміністратора
- `/admins` — список усіх адміністраторів
- `/users` — список усіх користувачів
- `/setdev` — змінити список розробників
- `/test` — тестовий режим
